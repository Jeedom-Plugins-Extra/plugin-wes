== Präsentation ==
Dieses Plugin ermöglicht es Ihnen die WES-Karten zu verwalten.

Zunächst wurde dieses Plugin erstellt, um sich mit WES zu verbinden.

=== Ersichtliche Daten auf dem Armaturenbrett : ===
* ''Die Temperatureingänge''
* ''Die Digitaleingänge''
* ''Die Relais''
* ''Die Impulszähler''
* "Das Zangenamperemeter"
* "Die Teleinfos"

== Installation/Einrichtung ==
Wir werden jetzt ein Gerät einrichten. Um dort hin zu kommen, klicken Sie auf ''Plugins /  Automatisierungsprotokolle / WES''.

Dann auf die Schaltfläche oben links "Gerät hinzufügen" klicken

Dann den Namen des Gerätes eingeben (z.B. WES 1)

Dann definieren :

* ''Eltern-Objekt''
* "Kategorie" (optional)
* "Aktivieren" (ankreuzen, andernfalls wird das Gerät nicht nutzbar sein)
* ''Sichtbar'' (optional, wenn Sie nicht wünschen, dass es auf dem Armaturenbrett sichtbar ist)
* '''Adresse IP'''
* ''Port''
* ''Konto''
* ''Passwort''

Alle Elemente der Karte finden Sie im Menü auf der linken Seite, durch Klicken auf dieses Symbol.  

=== Aktualisierungsrate  ===
Die Daten werden vom Plugin jede Minute gesammelt. Es ist nicht möglich, eine höhere Frequenz zu haben.
